Kontakt/Feedback

Diese Seite basiert auf dem Projekt "Meine Stadt Transparent" und bereitet die Daten des Ratsinformationssystem Münster auf.

Diese Seite wird betrieben von Code for Münster, einer Gruppe von Menschen aus Münster, die in ihrer Freizeit ehrenamtlich auf mehr Transparenz der Politik und Verwaltung hinwirken wollen.

Bei Fragen oder Anmerkungen lässt sich Code for Münster erreichen:

  • Per E-Mail: muenster@codefor.de
  • Auf Twitter @codeformuenster

"Meine Stadt Transparent" ist ein Open Source Projekt und kann unter Beachtung der MIT Lizenz frei modifiziert und kopiert werden.
